# linux下常用的命令快捷键

```
按使用频率顺序排序：
tab      命令补全工具
Ctrl + a 切换到命令行开始
Ctrl + e 切换到命令行末尾
Ctrl + c 终止当前命令或脚本           
Ctrl + d 退出当前shell，相当于exit。
Ctrl + l 清除屏幕内容，相当于clear。
Ctrl + u 清除光标之前的内容（还有剪切板的功能）
Ctrl + k 清除光标之后的内容
----------上面的命令必须熟练应用，不要背，多实践---------------
Ctrl + y 复制剪切板里的内容（如执行了Ctrl + u后）
Ctrl + h 类似于退格键（删除键）
Ctrl + r 在历史命令中查找 （非常好用，输入关键字就调出以前的命令了）
```
